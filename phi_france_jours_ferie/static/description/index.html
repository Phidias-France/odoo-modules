French holidays

Generating France public holidays for a calendar !

Informations

Public holiday are created by calling the webservice 'https://calendrier.api.gouv.fr/jours-feries'. The webservice returns public holidays for the current year and the next 5 years.

Usage

1. Access to calendars

  1. Switch to debug ode.
  2. Go to Settings -> Technical -> Working Times.

2. Generate public holiday for a calendar

  1. Open a calendar.
  2. In action menu, select Generate public holiday.

2. Select a zone

  1. Select a zone (Metropole, Guyane, Saint-Martin,...).
  2. Click on GENERATE button.

2. Check Time Off days in calendar

Select Global Time Off tab, the french holidays are created for the 5 next years.
